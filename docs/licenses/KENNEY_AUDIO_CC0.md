# Kenney audio assets

The runtime Foley files under `public/audio/foley/` are selected, renamed
copies from these official Kenney packs:

- **RPG Audio 1.0** — door, latch and cloth recordings.
- **Impact Sounds 1.0** — concrete/carpet footsteps and impact recordings.

Creator/distributor: Kenney (https://kenney.nl/)

License: Creative Commons Zero (CC0)
https://creativecommons.org/publicdomain/zero/1.0/

The source packs state that the assets may be used in personal, educational
and commercial projects. Attribution is not required, but the project credits
Kenney here for provenance.

Official source pages:

- https://kenney.nl/assets/rpg-audio
- https://kenney.nl/assets/impact-sounds

Downloaded for the Web runtime audio pass on 2026-07-23.
