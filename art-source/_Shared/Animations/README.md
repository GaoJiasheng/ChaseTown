Reference:
- `docs/02_Codex外包资产规格.md` §2.

Final files belong here:
- Shared Humanoid skeleton FBX.
- Animation FBX or clips.
- Preview and report files.

Required clips:
- `Idle`
- `Run`
- `Walk`
- `LookAround`
- `ScaredCaught`
- `Celebrate`
- `PointAlert`
- Optional: `TurnLeft`, `TurnRight`

Acceptance:
- Unity Humanoid Avatar validation is green.
- 30 fps baked animation.
- Root Motion off.
- Run/Walk are in-place and face +Z.
- Kid, Villain, and Police variants can share or retarget these clips.
